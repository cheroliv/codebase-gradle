**Statut** : TERMINE (2026-05-19, session 046)

composé de canaux convergents (EAGER/LAZY, RAG pgvector, Graphify, Ressources).
Le `CompositeContextBuilder` (déjà fait via EPIC 9) agrège ces canaux.

=== Les Quatre Canaux

[cols="1,2,3,3"]
|===
| Canal | Technologie | Type d'information | Rôle
| **EAGER/LAZY** | Fichiers AsciiDoc + `.agents/` | Règles, procédures, backlog, historique | Gouvernance déterministe
| **RAG pgvector** | LangChain4j + ONNX + PostgreSQL | Documents, code source, SPG, articles | Similarité vectorielle
| **Graphify** | `graph.json` + plantuml-gradle | Nœuds, arêtes, communautés, dépendances | Relations exactes
| **Ressources** | Dossiers encyclopédies, archives, datasets | Données brutes nettoyées, corpus, historique long | Mémoire froide
|===
