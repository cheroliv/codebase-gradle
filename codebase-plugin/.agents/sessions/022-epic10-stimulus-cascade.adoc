== Architecture du Pipeline

----
Brain Dump (texte ou .adoc)
    │
    ▼
parseSections() — détection headings == .+
    │
    ▼
classify() — US-9.14 VisionOpinionClassifier (Ollama deepseek-v4-pro)
    │
    ├── VISION → routeSection() → LLM routing (SystemMessage 4 docs racines)
    │           → DilutionTarget (document + section + rationale)
    │
    └── OPINION → confinement (dilutionTarget = null)
    │
    ▼
archiveReport() → configuration/vision-archive/$DATE_stimulus-dilution/
    ├── brain-dump-original.adoc
    ├── cascade-report.json
    └── cascade-report.adoc
    │
    ▼
exports → build/stimulus-reports/
    ├── stimulus-cascade.json
    └── stimulus-cascade.adoc
----
