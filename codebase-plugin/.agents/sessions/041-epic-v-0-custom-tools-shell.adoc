== Leçons

- koog `object` standalone n'accepte pas `companion object` → membres directs

- `ExecGradleTool` doit utiliser `user.dir` du vrai projet (pas `/tmp/workspace-root`)
- `SecurityException` (pas `IllegalArgumentException`) pour les rejets de sécurité
