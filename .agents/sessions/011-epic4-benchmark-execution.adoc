== Leçons

- LangChain4j AiServices multi-paramètres nécessite @V pour nommer chaque variable
- java.util.Arrays.toString() n'est pas un sérialiseur JSON
- Les longs contextes 128K tokens nécessitent timeout Ollama ≥ 300s
- Les 5 scénarios ont le même code path pour MVP (RAG/Graphify non câblés)
  → pas de mesure différentielle réelle. À cabler dans session 012.
- deepseek-v4-pro:cloud est très robuste sur la classification cercle de confiance
  même à 128K tokens de contexte filler. Les fixtures sont peut-être trop "évidentes".
