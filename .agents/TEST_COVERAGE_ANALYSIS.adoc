== Modules

=== codebase-gradle (src/main/kotlin/codebase/benchmark/)
- `BenchmarkRunner.kt` : runner + `buildContext()` RAG/Graphify + pgvector + SpatialPerceptionExpert LangChain4j + ContextFiller + BenchmarkFixtures
- `BenchmarkRunnerMain.kt` : entry point runBenchmark (passe pgJdbcUrl + graphJsonPath)
- `BenchmarkReportExporter.kt` : export JSON→AsciiDoc (parser JSON minimal, 0 dépendance externe)
- `BenchmarkReportExportMain.kt` : entry point exportBenchmarkReport
- `GraphContext.kt` : GraphModel/GraphNode/GraphEdge Jackson + GraphContextBuilder (load/neighborhood)
- `GraphGeneratorMain.kt` : scanner workspace → graph.json

=== codebase-gradle (buildSrc/.../benchmark/)
- `benchmark.kt` : data classes (TokenThreshold, BenchmarkScenario, BoundaryCrossingEvent, SpatialPerceptionResult, BenchmarkReport, BenchmarkConfig) + BenchmarkProtocol (160 lignes)

=== codebase-gradle (src/main/kotlin/codebase/rag/)
- `KotlinMetadataExtractor.kt` : extraction package_name via regex `^package\s+`, class_name via regex de dernier type (class/object/interface), fallback filename sans extension
- `VectorStore.kt` : schema documents +3 colonnes (package_name, class_name, repo_name), insertDocument avec metadata, findByPackage(), DocRecord data class
- `CodebaseIndexerMain.kt` : utilise KotlinMetadataExtractor, anonymisation YAML/JSON via YamlConfigAnonymizer
- `YamlConfigAnonymizer.kt` : anonymisation text-based YAML/JSON, matching par contenance (token/key/password/secret/credential/auth)

=== codebase-gradle (src/test/kotlin/codebase/scenarios/)
- `PgVectorTestContext.kt` : singleton partageant conteneur pgvector, modele ONNX, fileChunks, topResults
- `PgVectorContainerSteps.kt` : demarrage/arret conteneur pgvector + verification extension vector + creation table vector(384)
- `DatasetEmbeddingSteps.kt` : steps canon partages, schema synchro metadata, companion extractKotlinMetadata
- `WorkspaceWalkerSteps.kt` : steps walker + verification type kt/adoc/yml/yaml/json
- `MetadataIndexingSteps.kt` : steps specifiques metadata
- `ConfigIndexingSteps.kt` : steps anonymisation YAML/JSON + verification zero secret
- `CucumberTestRunner.kt` : @Suite JUnit Platform
- `TestCleanupExtension.kt` : nettoyage containers orphelins
